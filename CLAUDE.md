## Nombre del proyecto
`heyamigo-web`

## ¿Qué es / qué hace?
Página web de presentación (landing) de **Hey Amigo**, la plataforma para
migrantes. La web es el escaparate público; la app vive en otro dominio.

## Tipo
[x] Web / página
[ ] API / backend
[ ] Bot / automatización

## Tecnología principal
HTML + CSS + JavaScript en un solo archivo (`index.html`). Sin frameworks, fácil
de publicar en cualquier hosting. Trilingüe: español, inglés, francés.
